Easy generation and validation of passwords with one ore more of the following requirements:
- MinLength
- MaxLength
- AllowLowerCase
- RequireLowerCase
- AllowUpperCase
- RequireUpperCase
- AllowNumbers
- RequireNumber
- AllowSpecial
- RequireSpecial
- MaxConsecutiveSameCharacter
- DisableConfusableCharacters
- ForbiddenSequences
    -  e.g. dont allow username in password
